# Password Strength Analyzer

# Description
The Password Strength Analyzer is a Python-based security tool designed to evaluate the strength 
of user-entered passwords.This project checks different security factors such as password length,
uppercase and lowercase letters, numbers, and special characters to determine whether a password 
is weak, medium, or strong.

The main purpose of this project is to help users create secure passwords and understand the 
importance of password security in modern systems. The program also provides password strength 
percentages and suggests improvements for creating stronger passwords.

This project is useful for learning basic cybersecurity concepts, password validation techniques,
and Python programming fundamentals.

# Features
- Password length checking
- Uppercase and lowercase detection
- Number checking
- Special character checking
- Password strength percentage
- Weak, Medium, and Strong password classification
- Strong password suggestions
